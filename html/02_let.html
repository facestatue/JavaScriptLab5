Using let

When a variable is declared using the keyword let, it may be changed later. Using let does not allow you to redeclare a variable. The syntax for declaring a variable with let is:

  • Declare variable carName: let carName;
  • Declare variable carName and assign the value "Volvo": let carName = "Volvo";
  • Declare variable x: let x;
  • Declare variable x and assign the value 5: let x = 5;
