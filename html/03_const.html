Using const

When a variable is declared using the keyword const, it may not be changed later. Using const does not allow you to redeclare or change a variable. The syntax for declaring a variable with const is:

  • Declare variable carName: const carName;
  • Declare variable carName and assign the value "Volvo": const carName = "Volvo";
  • Declare variable x: const x;
  • Declare variable x and assign the value 5: const x = 5;
